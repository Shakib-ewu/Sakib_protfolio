The program computes the famous Mandelbrot fractal — for each pixel in a 2D image, it maps that pixel to a point in complex number space and iterates the formula z = z² + c to determine whether that point belongs to the Mandelbrot set. The number of iterations before "escaping" determines the pixel's color.
